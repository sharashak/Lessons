1. Что такое HTML?

        HTML (HyperText Markup Language) досл. "Язык разметки гипертекста" - система верстки,
    отвечающая за то, как всё должно выглядеть и располагаться на веб-странице.
        Нуждается в объявлении и имеет структурированную форму - некоторые элементы
    для HTML-файла являются обязательными (html, body, etc.)


2. Что такое CSS?

        CSS (Cascading Style Sheets) - это не язык разметки, а язык таблицы стилей. Так же, как и HTML,
    не является языком программирования. Это код, который используется для стилизации веб-страниц. 
    Он позволяет применять стили выборочно к элементам в документах HTML. CSS-код необходимо применять к HTML-документу.
    Иначе говоря, можно написать что-то на CSS, без применения это не даст никаких изменений.


3. Что такое JS?

        JavaScript - язык программирования, изначально предназначенный для того, чтобы сделать веб-страницы "живыми".
    Что-о-о? Предназначенный для того, чтобы размещать на странице динамически обновляемый контент. Программы на JS
    называются скриптами. Их можно встроить в HTML и они могут выполняться при загрузке веб-страниц. JS может выполняться
    на сервере, на клиентском устройстве и вообще на любом устройстве, которое имеет специальный JavaScript-движок.
    Разные браузеры имеют разные движки для JS.


4. Что такое PHP?

        PHP (Hypertext Preprocessor) - язык программирования общего назначения, специально сконструированный
    для веб-разработки. Также может внедряться непосредственно в HTML. Отличается от JavaScript тем,
    что PHP-скрипты выполняются на сервере и создают HTML, который посылается клиенту. Если на сервере расположен скрипт,
    конечный пользователь может даже не узнать, какой код был использован для получения результата его выполнения.
    Так же можно настроить всё таким образом, чтобы клиенты не поняли, получили они HTML-файл или результат выполнения скрипта.


5. Что такое MySQL?

        MySQL - реляционная система управления базами данных с моделью клиент-сервер. Слово "реляционная" означает,
        что база данных располагается в виде таблиц, которые взаимосвязаны. Принцип работы MySQL:
        1) MySQL создает базу данных для хранения и управления данными, которые определяют отношения каждой таблицы.
        2) Клиенты делают запросы в MySQL, вводя определенные команды SQL
        3) Приложение сервера ответит запрошенной информацией и появится на стороне клиента.
